# Emotion-Detection-in-Hinglish-Text_ML-CA3
Comparison of traditional ML, deep learning, and transformer models for emotion recognition on Hinglish (Hindi-English) text. Implements SVM, Bi-LSTM, and DistilBERT on the EmoHi-58K dataset and reports accuracy/F1 for all models. Includes full pipeline, saved results, and inference script.
